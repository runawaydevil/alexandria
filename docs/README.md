## System Overview

Alexandria is a static web application that enables infinite navigation through GitHub markdown content. It provides a clean, focused reading experience for exploring documentation across millions of public repositories without requiring authentication.

Key characteristics:
- Client-side only application
- No backend server required
- No user tracking or analytics
- Offline-capable through local caching
- Rate limit aware with intelligent fallbacks
